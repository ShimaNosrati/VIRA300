This Excel document uses VBA macros to control the Workspace Manager.
Pressing the "Say Hello" button launches the macro.
"Hello World" will be echoed in the Workspace Manager Command History.

VBA macros must be enabled in Excel in order to run this sample.

For more details, please refer to section 2, "Essential COM Principles 
and Programming Techniques", of the PolyWorks SDK Reference Guide in 
the documentation directory.

This sample code was developed and tested with 
Microsoft Excel 2013 and Microsoft Excel for Office 365 MSO (16.0.11001.20070).
